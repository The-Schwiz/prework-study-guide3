CLI Commands

  • cd: migrate to directory
  • pwd: print working directory
  • ls: list files and directories on computer
  • code .: opens VS code with working directory
  • cd ..: navigate backword one directory at a time
  • mkdir: make directory
  • touch: create a new file
  • open: open file-name
  • ls -al ~/.ssh: checks for existing ssh keys
  • ssh-keygen -t ed25519 -C "email": generate ssh key
  • eval "$(ssh-agent -s)": start ssh agent
  • ssh-add: add ssh key to ssh agent
  • ssh -T git@github.com: verify fingerprint matches GH's RSA pub. key fingerprint

Git

  • git clone "copied SSH value": clone GH repository to local machine - make sure cd is correct
  • git checkout -b feature/"feature-name": create a feature branch and switches to it
  • git add -A: stages changes - informs git that the changes will be added in the next commit
  • git commit -m "message": captures snapshot of previously staged changes
  • git pull origin main: pulls main branch of remote repo to local repo
  • git push origin feature/"feature-name": pushes local changes to remote repo
  • git status: checks what branch we are in
